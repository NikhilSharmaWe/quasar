# quasar

Quasar is a **VIDEO + TEXT** chat application written in Go.
- Docker image: [nikhilsharmawe/quasar](https://hub.docker.com/repository/docker/nikhilsharmawe/quasar/general)

### Run Locally

For running locally just run `docker compose up` with the [compose file](https://github.com/NikhilSharmaWe/quasar/blob/main/docker-compose.yaml) in repo.
